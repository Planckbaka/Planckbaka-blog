# Next.js 使用手册

## 概述

Next.js 是一个用于构建全栈 Web 应用程序的 React 框架。它提供了构建用户界面的 React 组件，以及额外的功能和优化。Next.js 还会自动配置底层工具如打包器和编译器，让开发者专注于产品构建和快速交付。

## 核心特性

### 1. 路由系统
- **App Router**：支持 React Server Components 的新路由器
- **Pages Router**：原始路由器，仍然支持并持续改进
- **文件系统路由**：基于文件结构自动生成路由
- **动态路由**：支持参数化路由

### 2. 渲染方式
- **服务端渲染 (SSR)**：在服务器上渲染页面
- **静态站点生成 (SSG)**：构建时预渲染页面
- **客户端渲染 (CSR)**：在浏览器中渲染
- **增量静态再生 (ISR)**：按需更新静态页面

### 3. 性能优化
- **自动代码分割**：按路由自动分割代码
- **图片优化**：内置图片优化组件
- **字体优化**：自动优化和加载字体
- **脚本优化**：优化第三方脚本加载

## 数据获取

### 1. Server Components (服务器组件)

```tsx
// app/posts/page.tsx
interface Post {
  id: number
  title: string
  content: string
}

async function getPosts(): Promise<Post[]> {
  const res = await fetch('https://api.example.com/posts', {
    // 缓存配置
    next: { revalidate: 3600 } // 1小时后重新验证
  })
  
  if (!res.ok) {
    throw new Error('Failed to fetch posts')
  }
  
  return res.json()
}

export default async function PostsPage() {
  const posts = await getPosts()
  
  return (
    <div>
      <h1>Blog Posts</h1>
      {posts.map(post => (
        <article key={post.id}>
          <h2>{post.title}</h2>
          <p>{post.content}</p>
        </article>
      ))}
    </div>
  )
}
```

### 2. Client Components (客户端组件)

```tsx
'use client'

import { useState, useEffect } from 'react'

interface User {
  id: number
  name: string
  email: string
}

export default function UserProfile() {
  const [user, setUser] = useState<User | null>(null)
  const [loading, setLoading] = useState(true)

  useEffect(() => {
    async function fetchUser() {
      try {
        const res = await fetch('/api/user')
        const userData = await res.json()
        setUser(userData)
      } catch (error) {
        console.error('Failed to fetch user:', error)
      } finally {
        setLoading(false)
      }
    }

    fetchUser()
  }, [])

  if (loading) return <div>Loading...</div>
  if (!user) return <div>User not found</div>

  return (
    <div>
      <h1>{user.name}</h1>
      <p>{user.email}</p>
    </div>
  )
}
```

import { NextRequest, NextResponse } from 'next/server'

// GET 请求
export async function GET(request: NextRequest) {
  const searchParams = request.nextUrl.searchParams
  const page = searchParams.get('page') || '1'
  
  try {
    const posts = await fetchPostsFromDatabase(parseInt(page))
    return NextResponse.json(posts)
  } catch (error) {
    return NextResponse.json(
      { error: 'Failed to fetch posts' },
      { status: 500 }
    )
  }
}

// POST 请求
export async function POST(request: NextRequest) {
  try {
    const body = await request.json()
    const newPost = await createPost(body)
    return NextResponse.json(newPost, { status: 201 })
  } catch (error) {
    return NextResponse.json(
      { error: 'Failed to create post' },
      { status: 500 }
    )
  }
}

// app/api/posts/[id]/route.ts - 动态 API 路由
export async function GET(
  request: NextRequest,
  { params }: { params: { id: string } }
) {
  const post = await getPostById(params.id)
  
  if (!post) {
    return NextResponse.json(
      { error: 'Post not found' },
      { status: 404 }
    )
  }
  
  return NextResponse.json(post)
}

## 中间件

```tsx
// middleware.ts
import { NextResponse } from 'next/server'
import type { NextRequest } from 'next/server'

export function middleware(request: NextRequest) {
  // 认证检查
  if (request.nextUrl.pathname.startsWith('/dashboard')) {
    const token = request.cookies.get('token')
    
    if (!token) {
      return NextResponse.redirect(new URL('/login', request.url))
    }
  }
  
  // 地理位置重定向
  const country = request.geo?.country || 'US'
  if (country === 'CN' && !request.nextUrl.pathname.startsWith('/cn')) {
    return NextResponse.redirect(new URL('/cn', request.url))
  }
  
  // 添加自定义头部
  const response = NextResponse.next()
  response.headers.set('x-custom-header', 'my-value')
  
  return response
}

export const config = {
  matcher: [
    /*
     * Match all request paths except for the ones starting with:
     * - api (API routes)
     * - _next/static (static files)
     * - _next/image (image optimization files)
     * - favicon.ico (favicon file)
     */
    '/((?!api|_next/static|_next/image|favicon.ico).*)',
  ],
}
```

### 2. 自定义服务器

```js
// server.js
const { createServer } = require('http')
const { parse } = require('url')
const next = require('next')

const dev = process.env.NODE_ENV !== 'production'
const hostname = 'localhost'
const port = 3000

const app = next({ dev, hostname, port })
const handle = app.getRequestHandler()

app.prepare().then(() => {
  createServer(async (req, res) => {
    try {
      const parsedUrl = parse(req.url, true)
      await handle(req, res, parsedUrl)
    } catch (err) {
      console.error('Error occurred handling', req.url, err)
      res.statusCode = 500
      res.end('internal server error')
    }
  }).listen(port, (err) => {
    if (err) throw err
    console.log(`> Ready on http://${hostname}:${port}`)
  })
})
```

## 总结

Next.js 是一个功能强大的 React 框架，提供了构建现代 Web 应用所需的所有工具和功能。通过合理使用其特性，可以构建出性能优异、SEO 友好的应用程序。

### 关键要点

1. **选择合适的渲染策略**：根据内容特性选择 SSR、SSG 或 CSR
2. **优化性能**：利用自动代码分割、图片优化等特性
